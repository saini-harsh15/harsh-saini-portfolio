# Harsh Saini – Developer Portfolio

A modern, responsive portfolio website showcasing my projects, skills, and backend development expertise. Built using Next.js with a focus on performance, clean UI, and real-world functionality.

---

## 🚀 Getting Started

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open http://localhost:3000 in your browser to view the application.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React, Tailwind CSS
* **Backend:** API Routes (Next.js), Node.js
* **Email Service:** Resend API
* **Tools:** Git, Vercel

---

## ✨ Features

* Fully responsive modern UI
* Interactive project showcase
* Contact form with email integration
* Clean and structured component architecture
* Optimized performance using Next.js

---

## 📁 Project Structure

```
app/            # Pages and routing
components/     # Reusable UI components
public/         # Static assets
lib/            # Utility functions
```

---

## ⚙️ Environment Variables

Create a `.env.local` file in the root directory and add:

```
RESEND_API_KEY=your_resend_api_key
MY_EMAIL=your_email_address
```

---

## 🌐 Deployment

The project is deployed using **Vercel**.

To deploy your own version:

1. Push the project to GitHub
2. Import the repository in Vercel
3. Add environment variables
4. Deploy
